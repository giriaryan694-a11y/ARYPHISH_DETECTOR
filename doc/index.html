ARYPHISH_DETECTOR

AI‑Powered Real‑Time Phishing Detection

View on GitHub

Overview

ARYPHISH_DETECTOR is an advanced hybrid‑AI tool that uses Google Gemini and OpenAI ChatGPT to analyze URLs and detect phishing attacks in real time. Unlike traditional URL filters, it fetches the live HTML of the target page and performs deep reasoning to identify hidden threats, obfuscation, and deceptive design elements.

Why This Tool?

Human vigilance fails. Databases get outdated. Attackers evolve faster than corporate training modules. ARYPHISH_DETECTOR fills this gap by providing:

  • Dual‑AI verification
  • Live content analysis
  • Detection of deceptive login forms
  • Typosquatted domains and brand impersonation
  • Obfuscated JavaScript traps

How It Works

Frontend

A clean, responsive interface built with Tailwind CSS. Users enter a URL, select an AI model, and view the results in styled cards.

Backend

Flask handles async URL fetching via httpx and passes the raw HTML to Gemini or ChatGPT for structured JSON analysis.

Features

  • Dual AI mode (Gemini + ChatGPT)
  • Single model mode
  • Real-time HTML fetching
  • Color-coded verdicts
  • JSON-based reasoning from AI
  • Stylish CLI banner using pyfiglet
